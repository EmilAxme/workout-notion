## Что создаётся

Скрипт автоматически генерирует в Notion:

- **Главная страница** — описание целей и сплита
- **БД «Тренировки»** — 48 записей (12 недель × 4 дня)
- **БД «Упражнения»** — 288 записей с relation к тренировкам, по 6 упражнений на день
- **БД «Прогресс тела»** — вес, % жира, замеры
- **БД «Личные рекорды»** — рабочий вес и лучшие повторы

Перед созданием скрипт ищет старую систему тренировок в том же родителе и **предлагает архивировать её** (`archived=true` — обратимо, через Trash в Notion UI).

## Быстрый старт

```bash
git clone https://github.com/EmilAxme/workout-notion.git
cd workout-notion
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python setup_notion_workout.py
```

Скрипт запросит:
1. **Notion Integration Token** — создай на [notion.so/my-integrations](https://www.notion.so/my-integrations)
2. **Parent Page ID** — ID/URL страницы, где будет создана система

> Не забудь подключить интеграцию к странице: **Share → Invite → выбери интеграцию**

Или задай через переменные окружения:
```bash
cp .env.example .env
# заполни .env своими ключами
python setup_notion_workout.py
```

## Программа

**Цель:** РЕКОМПОЗИЦИЯ — удерживаем вес ~80 кг, минус 2-4 кг жира.
**Питание:** ~2400 ккал, белок 180 г. На неделе 7 — диета-брейк (~2790 ккал на поддержке).
**Прогрессия:** двойная по повторам. План по неделям одинаковый — в дефиците силы расти на расписании смысла нет. Заполняй `Вес_факт` и `Повторы_факт` по факту.

| День | Группы мышц | Упр. |
|---|---|---|
| Пн | Грудь + Трицепс | 6 |
| Вт | Спина + Бицепс | 6 |
| Чт | Ноги | 6 |
| Пт | Плечи + Руки | 6 |

### Шаблон тренировок

**Пн (Грудь+Трицепс):** Жим штанги лёжа 4×6-8 · Жим гантелей на наклонной 4×8-10 · Жим в тренажёре/брусья 3×10-12 · Сведения в кроссовере 3×12-15 · Трицепс на блоке (канат) 4×10-12 · Французский жим 3×10-12

**Вт (Спина+Бицепс):** Подтягивания 4×6-10 · Тяга штанги в наклоне 4×8-10 · Тяга верхнего блока 3×10-12 · Тяга гантели одной рукой 3×10-12 · Подъём штанги на бицепс 4×8-10 · Молотки 3×10-12

**Чт (Ноги):** Присед 4×8-10 · Жим ногами 3×12-15 · Румынская тяга 3×10-12 · Болгарские выпады 3×10-12 · Сгибания ног лёжа 3×12-15 · Икры стоя 4×15-20

**Пт (Плечи+Руки):** Жим гантелей/штанги сидя 4×8-10 · Махи в стороны 4×12-15 · Махи в наклоне 3×15 · Скамья Скотта 4×10-12 · Трицепс обратным хватом 4×10-12 · Суперсет молоток+бицепс 3×12

## Как пользоваться

### После тренировки
1. Открой БД «Тренировки», найди сегодняшнюю
2. Поставь **Статус → Выполнено**, отметь самочувствие, сон, утренний вес
3. В связанных упражнениях заполни: `Вес_факт`, `Повторы_факт`, `Выполнено ✅`

### Прогрессия (двойная)
Если в подходе сделал верх диапазона на всех подходах — на следующей тренировке добавь подход / +2.5 кг. Не дотянул — повторяй с тем же весом.

### Еженедельно
- Утром в Пн измерь вес — в «Прогресс тела» или прямо в поле `Вес_утром_кг` тренировки
- Раз в 2-4 недели — замеры (талия, грудь, бицепс, бедро)

## Архивация старой системы

При запуске скрипт ищет в parent'е страницы с ключевыми словами `«Программа тренировок»`, `«Рекомпозиция»`, `«Тренировок»`, `«Workout»` и предлагает заархивировать. Архивация в Notion — это **обратимое** перемещение в Trash, восстанавливается через UI (`...` → Restore).
